# 🐦 Django Tweet Project

A simple Twitter-like web application built using Django where users can register, login, post tweets, upload images, and search tweets.

---

## 🚀 Features

- 🔐 User Authentication (Register, Login, Logout)
- ✍️ Create Tweets (max 240 characters)
- 🖼️ Upload Images with Tweets
- 🔍 Search Tweets by text or username
- 🕒 Timestamp (created & updated)
- 🎨 Responsive UI using Bootstrap

---

## 🛠️ Tech Stack

- Backend: Django
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite (default)
- Language: Python

---


---

## 🔍 Search Functionality

Users can search:
- Tweet text
- Username

---

## 📸 Media Handling

- Uploaded images are stored in `media/`
- Make sure `MEDIA_URL` and `MEDIA_ROOT` are configured

---

## ⚠️ Important Notes

- `db.sqlite3` is not included (use migrations)
- `media/` folder is ignored in GitHub
- Use `.gitignore` to avoid unnecessary files
